# enukasathmina.github.io

This repository contains a static portfolio website for **Enuka Sathmina** built with HTML, CSS, and JavaScript.

## Files

- `index.html` — main website entry point
- `styles.css` — responsive site styling
- `script.js` — small script for dynamic content

## Deploying to GitHub Pages

1. Push this repository to GitHub under the username `enukasathmina`.
2. In the repository settings, open the **Pages** section.
3. Set the source to the `main` branch and the root folder `/`.
4. Save and wait a few minutes. Your site will be available at `https://enukasathmina.github.io`.

## Customize

- Update the text in `index.html` with your real bio, projects, and contact details.
- Replace sample project cards with actual links or screenshots.
- Adjust the color palette and layout in `styles.css` to match your branding.
